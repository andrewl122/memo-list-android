Memo List Android App

Memo List is an Android application that allows users to create, organize, and manage personal memos with priority and date tracking. The application provides a structured interface for quickly adding, editing, deleting, and searching memos while maintaining persistent local storage.

Project Type

Team Project — developed collaboratively as part of a mobile application development course.

My Contributions

Implemented memo creation, editing, and deletion functionality

Integrated SQLite database storage for persistent memo management

Developed filtering and sorting features using SharedPreferences

Assisted with UI development and debugging during testing

Features

Create, edit, and delete memos

Assign priority levels to memos

Track memo creation dates

Search and filter memo entries

Persist memo data locally using SQLite

Interactive UI with dynamic filtering and sorting

Tech Stack

Java

Android SDK

SQLite

SharedPreferences

Android Studio

Purpose

The goal of this project was to explore Android mobile development concepts including local database management, multi-screen application navigation, and user interface design. The application demonstrates how structured data can be stored, retrieved, and manipulated within a mobile environment.
