# Memo List

Memo List is an Android application that allows users to create, organize, and manage personal memos with priority and date tracking. The app demonstrates core Android development concepts including multi-screen navigation, persistent local data storage, and interactive user interface components.

## Project Type

**Team Project** — developed collaboratively as part of a mobile application development course.

### My Contributions

- Implemented **memo creation, editing, and deletion functionality**
- Integrated **SQLite database storage** for persistent memo management
- Developed **filtering and sorting functionality** using SharedPreferences
- Assisted with **UI development and debugging during testing**

## Features

- Create, edit, and delete memos
- Assign priority levels and track memo dates
- Search and filter memo entries
- Persist memo data locally using SQLite
- Interactive UI with real-time filtering and sorting

## Tech Stack

- Java
- Android SDK
- SQLite
- SharedPreferences
- Android Studio

## Purpose

The goal of this project was to explore Android mobile development concepts including local database management, multi-screen navigation, and structured data handling. The application demonstrates how mobile systems store, retrieve, and manipulate user data efficiently within a local database environment.
